Sassafras

Simple aliases for Discord

How does it work?

Once you begin hosting Sassafras, typing an allias anywhere in a message will have it replaced by the translation phrase.

This will happen no matter where you do it, other computers, desktops, laptops, even on mobile!

How to use the Alias File

The alias file is simple json, so it's easy for anyone to edit and understand with ease.

{
  ";;ping": "pong",
  ";;rules": "1). don't talk about fight club, 2). DON'T talk about fight club",
  ";;parallax": "Parallax is a displacement or difference in the apparent position of..."
}

Everything has to be quoted, and a comma goes after every line except the last one. If you only have one line, you will have no commas.
